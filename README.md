# Golden Jubilee Invite

Standalone static site for the 50th Marriage Anniversary (Golden Jubilee) of Lala Jai Prakash Gupta & Uma Rani Gupta.

## Structure
- index.html — invitation page
- styles.css — theme styles
- script.js — animations (GSAP via CDN)
- background-1.jpeg, background-2.jpeg — section backgrounds

## Local Preview
Open index.html in your browser (no build needed).

## Deploy
- Initialize git: `git init && git add . && git commit -m "init"`
- Push to any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages).

Optional: add a `music.mp3` in the project root to enable background audio when opening the invite. If omitted, the page works without audio.
